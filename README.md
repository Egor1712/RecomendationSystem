[Датасет](https://www.kaggle.com/competitions/h-and-m-personalized-fashion-recommendations/data) для работы

Перед запуском:
Датасет лучше положить в папку ../dataset вне репозитория. Если хочется поменять пути, то все они инициализируется в файле consts.py


Так как изначальный датасет достаотчно большой, можно запустить файл dataset_reducing.py - он значительно уменьшит датасет

Пока реализовано 3 модели - CatBoost, LightFm, TwoTower, все файлы распололжены в соответствующих папках

После обучения LightFm можно запустить api.py для поднятия апи для взаимодействия
